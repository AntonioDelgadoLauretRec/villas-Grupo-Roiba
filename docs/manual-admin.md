## 1. Acceso al Panel

### Cómo entrar

1. Abre tu navegador de internet (Chrome, Safari, Firefox...) y escribe en la barra de direcciones: **tudominio.com/admin**
2. Verás una pantalla de acceso. Escribe tu **dirección de email** y tu **contraseña**
3. Pulsa el botón **"Acceder"**

> **No recuerdo mi contraseña**: Contacta con el administrador principal del sistema para que te genere una nueva.

### Cómo salir

- En la barra lateral izquierda, al final del todo, pulsa **"Cerrar sesión"**
- Tu sesión se cerrará y volverás a la pantalla de acceso

### Seguridad

- No compartas tus datos de acceso con terceros
- Si sospechas que alguien ha accedido a tu cuenta, contacta con el administrador inmediatamente
- La sesión se cierra automáticamente tras un periodo de inactividad

## 4. Gestionar Villas

Las villas son las propiedades que forman el portfolio de la empresa.

### Ver la lista de villas

1. Pulsa **"Villas"** en el menú lateral
2. Verás una lista con todas las villas, mostrando su nombre, ubicación y estado

```
┌──────────────────────────────────────────────────────────────────┐
│  Villas                                        [ + Nueva villa ] │
│──────────────────────────────────────────────────────────────────│
│                                                                  │
│  Nombre              Ubicación       Estado       Acciones       │
│  ─────────────────────────────────────────────────────────────── │
│  Villa Aurora        Cap Cana        Publicada    👁  ✏️  🗑     │
│  Villa Excellence    Bávaro          Publicada    👁  ✏️  🗑     │
│  Villa Oceano        Cap Cana        Borrador     👁  ✏️  🗑     │
│                                                   │   │   │      │
│                                                   │   │   └─ Eliminar
│                                                   │   └─ Editar  │
│                                                   └─ Mostrar/    │
│                                                      Ocultar     │
└──────────────────────────────────────────────────────────────────┘
```

### Crear una nueva villa

1. Pulsa el botón **"Nueva villa"** (arriba a la derecha)
2. Rellena los campos del formulario:

| Campo | Qué escribir | Obligatorio |
|-------|-------------|-------------|
| **Título** | Nombre de la villa (ejemplo: "Villa Aurora") | Sí |
| **Slug** | Identificador corto para la dirección web. Solo letras minúsculas, números y guiones. Ejemplo: `villa-aurora-cap-cana` | Sí |
| **Ubicación** | Zona donde se encuentra (ejemplo: "Cap Cana") | No |
| **Año** | Año de construcción o entrega | No |
| **Descripción corta** | Resumen en 1-2 frases | No |
| **Descripción larga** | Texto completo con todos los detalles | No |
| **Habitaciones** | Número de dormitorios | No |
| **Baños** | Número de baños | No |
| **Superficie** | Metros cuadrados construidos | No |
| **Precio** | Precio en dólares (USD) | No |
| **Imagen principal** | Foto principal de la villa | No |
| **Orden** | Un número que controla la posición en la lista (los números más bajos aparecen primero) | No |
| **Estado** | "Publicada" para que sea visible en la web, "Borrador" para guardar sin publicar | No |
| **Destacada** | Si quieres que aparezca en posición preferente | No |

3. Pulsa **"Crear villa"**

### Editar una villa

1. En la lista de villas, pulsa el icono del **lápiz** (✏️) de la villa que quieras modificar
2. Cambia los campos que necesites
3. Pulsa **"Guardar cambios"**

### Mostrar u ocultar una villa

- Pulsa el icono del **ojo** (👁) junto a la villa
  - Si la villa está visible, pasará a estar oculta
  - Si está oculta, pasará a estar visible

```
  Estado actual          Pulsas 👁          Nuevo estado
  ─────────────          ──────────          ────────────
  ✅ Publicada    ──→    se oculta    ──→   ❌ Borrador
  ❌ Borrador     ──→    se publica   ──→   ✅ Publicada
```

### Eliminar una villa

1. Pulsa el icono de la **papelera** (🗑) junto a la villa
2. Aparecerá un mensaje pidiendo confirmación
3. Pulsa **"Eliminar"** para borrarla definitivamente o **"Cancelar"** para mantenerla

```
 ┌────────────────────────────────────────┐
 │                                        │
 │   ⚠️  ¿Eliminar esta villa?            │
 │                                        │
 │   Esta acción no se puede deshacer.    │
 │                                        │
 │        [ Cancelar ]  [ Eliminar ]      │
 │            ↑              ↑            │
 │        Mantener la     Borrarla        │
 │        villa           para siempre    │
 │                                        │
 └────────────────────────────────────────┘
```

> **Importante**: Eliminar una villa es irreversible. Se borrarán también todas sus fotografías asociadas.

### Galería de imágenes

Al editar una villa, debajo del formulario principal encontrarás la **galería de imágenes**:
- **Añadir**: Arrastra una imagen o pulsa para seleccionar un archivo de tu ordenador
- **Eliminar**: Pasa el cursor sobre la imagen y pulsa "Eliminar"

## 5. Gestionar Proyectos

Los proyectos representan desarrollos urbanísticos futuros o en construcción. Funcionan de forma similar a las villas.

### Campos específicos de los proyectos

| Campo | Qué escribir |
|-------|-------------|
| **Título y Slug** | Igual que las villas |
| **Ubicación** | Zona del proyecto |
| **Fecha estimada** | Cuándo se espera que esté terminado (texto libre, ej.: "Q4 2026") |
| **Descripción** | Texto descriptivo del proyecto |
| **Estado** | Opciones disponibles (ver tabla siguiente) |

### Estados de los proyectos

| Estado | Significado | Visible en la web |
|--------|------------|-------------------|
| **Publicado** | Proyecto completado y visible | Sí |
| **Futuro** | Proyecto planificado, aún no iniciado | Sí |
| **En desarrollo** | Proyecto en fase de construcción | Sí |
| **Oculto** | Archivado, no se muestra | No |

### Crear, editar, ocultar y eliminar

Las acciones funcionan exactamente igual que en las villas: botones de lápiz (editar), ojo (mostrar/ocultar) y papelera (eliminar).

## 6. Servicios

Los servicios son las actividades profesionales que ofrece la empresa. Aparecen en dos páginas de la web:
- **Página de inicio**: Los servicios principales con foto
- **Página de servicios**: El catálogo completo

### Cómo crear un servicio

1. Ve a **"Servicios"** en el menú lateral
2. Pulsa **"Nuevo servicio"**
3. Rellena los campos:

| Campo | Qué escribir |
|-------|-------------|
| **Slug** | Identificador corto (ejemplo: `diseno-arquitectonico`) |
| **Título** | Nombre del servicio (ejemplo: "Diseño Arquitectónico") |
| **Descripción** | Breve texto explicativo |
| **URL de imagen** | Dirección web de la imagen del servicio |
| **Página** | Elige dónde aparecerá: **Homepage** (página de inicio) o **Servicios** (página de servicios) |
| **Orden** | Posición en la lista (números bajos = primero) |
| **Visible** | Marca la casilla para que aparezca en la web |

4. Pulsa **"Crear"**

### Editar, ocultar y eliminar

Usa los iconos de cada servicio en la lista: lápiz para editar, ojo para mostrar/ocultar, papelera para eliminar.

## 7. Sub-servicios

Los sub-servicios son descripciones detalladas que aparecen en la página de servicios. Cada uno incluye un listado de lo que incluye.

### Campos del sub-servicio

| Campo | Qué escribir |
|-------|-------------|
| **Slug** | Identificador corto (ejemplo: `villas-premium`) |
| **Etiqueta** | Categoría corta (ejemplo: "Premium") |
| **Título** | Nombre completo del sub-servicio |
| **Descripción** | Texto explicativo detallado |
| **Incluye** | Lista de lo que abarca el servicio. **Escribe cada elemento en una línea separada** |
| **Nota adicional** | Texto complementario opcional |
| **Orden** | Posición en la lista |
| **Visible** | Marca la casilla para que aparezca en la web |

### Cómo rellenar el campo "Incluye"

Escribe cada elemento del servicio en una línea nueva, como si fuera una lista. Por ejemplo:

```
Diseño arquitectónico personalizado
Interiorismo integral
Selección de materiales premium
Gestión de permisos
```

Cada línea se convertirá en un punto de la lista visible en la web.

```
  Lo que escribes en el campo "Incluye":       Cómo se ve en la web:

  ┌────────────────────────────────────┐       ┌──────────────────────────┐
  │ Diseño arquitectónico personalizado│       │                          │
  │ Interiorismo integral              │  ──→  │  • Diseño arquitectónico │
  │ Selección de materiales premium    │       │    personalizado         │
  │ Gestión de permisos                │       │  • Interiorismo integral │
  └────────────────────────────────────┘       │  • Selección de          │
    ↑ Una línea = un elemento                  │    materiales premium    │
    ↑ NO uses guiones ni puntos                │  • Gestión de permisos   │
    ↑ Pulsa "Enter" para nueva línea           └──────────────────────────┘
```

## 8. Proceso de Trabajo

El proceso describe los pasos que sigue la empresa para llevar a cabo un proyecto. Aparece tanto en la página de inicio como en la página de proceso.

### Campos de cada paso

| Campo | Qué escribir |
|-------|-------------|
| **Número** | Número del paso (ejemplo: "01", "02") |
| **Título** | Nombre del paso (ejemplo: "Análisis de Terreno") |
| **Subtítulo** | Frase corta complementaria |
| **Descripción** | Texto explicativo del paso |
| **Detalle / Entregables** | Lo que se entrega al completar este paso. Usa el separador " · " para separar elementos (ejemplo: "Estudio técnico · Normativa urbanística · Criterios constructivos") |
| **Duración estimada** | Tiempo aproximado (ejemplo: "2-4 semanas") |
| **Orden** | Posición en la secuencia |
| **Visible** | Marca la casilla para que aparezca en la web |

## 13. Puntos de Interés (Mapa)

Los POIs (Puntos de Interés) son las ubicaciones que aparecen en el mapa interactivo de la página "Por qué Punta Cana". Representan resorts, playas, campos de golf, marinas y otros lugares relevantes.

### Campos de cada punto

| Campo | Qué escribir |
|-------|-------------|
| **Nombre** | Nombre del lugar (ejemplo: "Cap Cana Marina") |
| **Descripción** | Breve texto descriptivo |
| **Categoría** | Tipo de lugar. Opciones: Resort, Playa, Golf, Marina, Aeropuerto, Ocio, Naturaleza, Gastronomía |
| **Latitud** | Coordenada geográfica norte-sur (ejemplo: 18.5). Puedes obtenerla de Google Maps |
| **Longitud** | Coordenada geográfica este-oeste (ejemplo: -68.3). Puedes obtenerla de Google Maps |
| **Orden** | Posición en la lista |
| **Visible** | Marca la casilla para que aparezca en el mapa |

### Cómo obtener las coordenadas de un lugar

1. Abre **Google Maps** en tu navegador (maps.google.com)
2. Busca el lugar o haz clic derecho en el punto exacto del mapa
3. Se mostrarán dos números: el primero es la **latitud** y el segundo la **longitud**
4. Copia esos números en los campos correspondientes del formulario

```
  Google Maps: Clic derecho en el mapa
  ─────────────────────────────────────

  ┌─────────────────────────────────────────────┐
  │                                             │
  │        🗺️  (mapa de Punta Cana)             │
  │                                             │
  │                   📍 ← Clic derecho aquí    │
  │               ┌────────────────┐            │
  │               │ 18.5134, -68.3695│ ← Estos  │
  │               │ ──────────────── │   son los │
  │               │ Indicaciones...  │   números │
  │               │ ¿Qué hay aquí?  │   que     │
  │               └────────────────┘   necesitas │
  │                                             │
  └─────────────────────────────────────────────┘

  El PRIMER número (18.5134)  → Campo "Latitud"
  El SEGUNDO número (-68.3695) → Campo "Longitud"

  ⚠️  Importante: el número de longitud en Punta Cana
     siempre empieza con signo negativo (-)
```

### Ejemplo práctico

Para añadir la Marina de Cap Cana como punto de interés:

```
  Nombre:      Cap Cana Marina
  Categoría:   Marina
  Latitud:     18.5134
  Longitud:    -68.3695
                 ↑
                 No olvides el signo negativo
```

Esta sección permite editar los textos generales de la web que no pertenecen a ninguna sección específica: las imágenes del encabezado, las estadísticas, el texto "Sobre nosotros" de la página de inicio, los textos del pie de página, etc.

### Cómo acceder

1. Ve a **"Contenido global"** en el menú lateral (sección Configuración)
2. En el lado izquierdo verás una lista de bloques de contenido
3. Pulsa sobre el que quieras editar

### Bloques disponibles

| Bloque | Qué controla |
|--------|-------------|
| **Hero** | Los textos principales del encabezado de la página de inicio (título, subtítulo, botón) |
| **Imágenes Hero** | Las fotografías que se van alternando en el encabezado de la página de inicio |
| **Estadísticas** | Los números que aparecen en la barra de estadísticas (ej.: "15+ Proyectos entregados") |
| **Sección Nosotros** | Los párrafos de la sección "Quiénes Somos" de la página de inicio |
| **Contacto** | Email, teléfono, Instagram, dirección |
| **SEO** | Título y descripción del sitio para buscadores (Google) |
| **Enfoque** | Los puntos de la lista "Nuestro enfoque" en la página Nosotros |
| **Footer** | La descripción de la empresa y enlaces a redes sociales del pie de página |

### Cómo editar

1. Selecciona el bloque en la lista de la izquierda
2. Verás el contenido actual en un cuadro de texto en formato técnico (JSON)
3. Modifica **solo los textos entre comillas** (`"texto"`). No borres las llaves `{}`, corchetes `[]`, ni las comas
4. Pulsa **"Guardar"**

```
  Anatomía del formato JSON — Qué puedes cambiar y qué no:

  {                          ← NO TOCAR (llave de apertura)
    "value": "15+",          ← ✅ Puedes cambiar "15+" por otro texto
    "label": "Proyectos"     ← ✅ Puedes cambiar "Proyectos"
  }                          ← NO TOCAR (llave de cierre)
   ↑        ↑        ↑
   │        │        └── Las comillas "" son obligatorias
   │        └── Los dos puntos : son obligatorios
   └── El nombre del campo NO se cambia

  Resumen:
  ┌──────────────────────────────────────────────┐
  │  ✅ PUEDES cambiar:  Los textos entre ""     │
  │  ❌ NO toques:       {} [] , : ni los        │
  │                      nombres de los campos   │
  └──────────────────────────────────────────────┘
```

### Ejemplo: cambiar una estadística

Si el contenido muestra:

```
[
  {"value": "15+", "label": "Proyectos entregados"},
  {"value": "98%", "label": "Clientes satisfechos"}
]
```

Para cambiar "15+" por "20+", simplemente sustituye el texto:

```
[
  {"value": "20+", "label": "Proyectos entregados"},
  {"value": "98%", "label": "Clientes satisfechos"}
]
```

> **Precaución**: Si al guardar aparece un error de "JSON inválido", revisa que no falten comillas, comas o llaves. En caso de duda, contacta con el equipo técnico.

## 17. Subida de Imágenes

### Formatos admitidos

| Formato | Cuándo usarlo |
|---------|--------------|
| **JPG** | Para fotografías (el más habitual) |
| **PNG** | Para imágenes con fondo transparente |
| **WebP** | Formato moderno que ocupa menos espacio |
| **AVIF** | Formato de última generación, muy ligero |

### Tamaño máximo

- **10 MB** por imagen

### Recomendaciones

- Usa imágenes de al menos **1920 x 1080 píxeles** para que se vean bien en pantallas grandes
- Las fotografías en formato **horizontal** (apaisado) funcionan mejor como imágenes principales
- Antes de subir, optimiza las fotos para que la web cargue rápido. Puedes usar herramientas gratuitas como **TinyPNG** (tinypng.com) o **Squoosh** (squoosh.app)
- Las fotografías profesionales generan mayor confianza en los visitantes

### Cómo proporcionar la URL de una imagen

Para las secciones que piden una **"URL de imagen"** (servicios, equipo, valores, etc.), necesitas que la imagen esté alojada en internet. Opciones:

1. **Usar el almacenamiento propio**: Sube la imagen desde la sección de Villas o Proyectos y copia la dirección resultante
2. **Usar un servicio externo**: Sube la imagen a un servicio como Unsplash, Cloudinary o similar, y copia su dirección web

## 18. Conceptos Clave

| Término | Qué significa |
|---------|---------------|
| **Slug** | Un identificador corto y legible que forma parte de la dirección web (ver diagrama abajo) |
| **Orden** | Un número que determina la posición de un elemento en la lista (ver diagrama abajo) |
| **Visible / Publicado** | Una casilla que controla si el contenido aparece en la web pública. Si la casilla está desmarcada, el contenido solo es visible para los administradores |
| **URL** | La dirección de internet de una página o imagen (ejemplo: `https://ejemplo.com/foto.jpg`) |
| **SEO** | "Search Engine Optimization" (Optimización para Buscadores). Son los textos que controlan cómo aparece la web en los resultados de Google |
| **JSON** | Un formato técnico para almacenar datos estructurados. Se usa en la sección de "Contenido global". Se reconoce por el uso de llaves `{}`, corchetes `[]` y comillas `""` |
| **Caché** | Un sistema que guarda una copia temporal de la web para que cargue más rápido. Por eso los cambios pueden tardar un tiempo en reflejarse (hasta 1 hora) |

### Cómo funciona el Slug (ejemplo visual)

```
  Título de la villa:  "Villa Aurora en Cap Cana"
                                │
                                ▼
  Slug que escribes:   villa-aurora-cap-cana
                                │
                                ▼
  Dirección web resultante:

  https://gruporoiba.com / villas / villa-aurora-cap-cana
  ├───────────────────┤   ├─────┤   ├────────────────────┤
     Dominio del         Sección      Tu slug aparece
     sitio web                         aquí

  Reglas del slug:
  ┌─────────────────────────────────────────────────┐
  │  ✅  villa-aurora          (minúsculas + guiones)│
  │  ✅  villa-2025            (números permitidos)  │
  │  ❌  Villa Aurora          (sin mayúsculas)      │
  │  ❌  villa aurora          (sin espacios)        │
  │  ❌  villa_aurora          (sin guiones bajos)   │
  │  ❌  villa/aurora          (sin barras)          │
  └─────────────────────────────────────────────────┘
```

### Cómo funciona el Orden (ejemplo visual)

```
  Si asignas estos números de orden...

  Servicio A  →  orden: 10
  Servicio B  →  orden: 20
  Servicio C  →  orden: 30

  ...se mostrarán así en la web:

  1. Servicio A
  2. Servicio B
  3. Servicio C

  ¿Necesitas añadir uno nuevo entre A y B?
  Simplemente dale orden 15:

  Servicio A      →  orden: 10    →  1. Servicio A
  Servicio NUEVO  →  orden: 15    →  2. Servicio NUEVO
  Servicio B      →  orden: 20    →  3. Servicio B
  Servicio C      →  orden: 30    →  4. Servicio C

  💡 Consejo: Usa siempre múltiplos de 10 (10, 20, 30...)
     para dejar espacio entre elementos.
```

## 19. Consejos y Buenas Prácticas

1. **Guarda con frecuencia**: Después de hacer cambios importantes, pulsa "Guardar" antes de seguir editando
2. **Usa imágenes de calidad**: Las fotografías profesionales son la primera impresión del visitante. Evita imágenes borrosas o de baja resolución
3. **Slugs descriptivos**: Usa identificadores que describan el contenido. Ejemplo: `villa-excellence-bavaro` es mejor que `villa-1`
4. **Numeración con holgura**: Al asignar el orden de aparición, usa números como 10, 20, 30... Así podrás insertar nuevos elementos (15, 25...) sin tener que cambiar los demás
5. **Comprueba en la web**: Después de publicar contenido, visita la web pública para verificar que todo se ve correctamente. Usa el enlace **"Ver web pública"** en el menú lateral
6. **Textos concisos**: Los textos breves y directos funcionan mejor en la web. Evita párrafos excesivamente largos
7. **Un cambio a la vez**: Cuando estés aprendiendo el panel, haz un cambio, guárdalo, comprueba el resultado en la web, y después pasa al siguiente
8. **Cuidado con eliminar**: Borrar contenido es irreversible. Antes de eliminar, considera si es mejor simplemente ocultarlo (desmarcar la casilla "Visible")

## 20. Solución de Problemas

| Problema | Qué hacer |
|----------|-----------|
| **No puedo iniciar sesión** | Verifica que el email y contraseña sean correctos. Si sigues sin poder acceder, contacta al administrador principal |
| **Los cambios no se ven en la web** | Los cambios pueden tardar hasta 1 hora en aparecer por la caché. Si tras ese tiempo siguen sin verse, contacta al equipo técnico |
| **Error al guardar contenido global** | Revisa que el formato del texto sea correcto. Asegúrate de no haber borrado comillas, llaves o comas. Si tienes dudas, contacta al equipo técnico |
| **La imagen no aparece** | Comprueba que la URL de la imagen sea correcta copiándola en tu navegador. Si no se ve, la imagen puede haber sido eliminada o la dirección puede estar mal escrita |
| **No encuentro una villa o contenido** | Puede estar oculto (casilla "Visible" desmarcada). La lista del panel muestra todos los elementos, incluidos los ocultos (aparecen con la etiqueta "Oculto") |
| **He borrado algo por error** | Contacta inmediatamente con el equipo técnico. Algunos datos pueden recuperarse de las copias de seguridad |
| **El menú lateral no aparece** | En pantallas pequeñas (móvil/tablet), pulsa el icono de tres líneas (☰) en la esquina superior izquierda |
| **No sé qué coordenadas poner en un POI** | Busca el lugar en Google Maps, haz clic derecho sobre el punto y copia los dos números que aparecen (latitud y longitud) |
